# daily_automate

Personal automation daemon for daily engineering workflows.

## Features (planned)

- Daily standup generation
- PR management (CI monitoring, auto-reply drafts)
- PR review loop
- Slack digests
- JIRA automation
- Morning reminders & periodic nudges
- Metrics monitoring
- Weekly summary (Google Doc)

## Quick Start

```bash
./setup.sh
.venv/bin/python server.py run
```

Open http://localhost:8080
